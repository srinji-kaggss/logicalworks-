# Universal Run Spine

Date: 2026-06-07
Status: active bridge layer

## Overview

The Universal Run Spine provides a single machine-first index for linking deterministic artifacts
across `lgwks` subsystems (Axiom, Substrate, Sessions, etc.). It enables agents to navigate
verified work without path inference.

## Schema: `lgwks.run_index.v0`

- `run_id`: `run-<uuid>`
- `source`: The originating subsystem (e.g. `axiom`)
- `artifacts`: List of `{kind, path, schema, cid}`
- `links`: Directed relationships between artifacts (e.g. `described_by`)

## Verbs

`lgwks run adopt-axiom <dir>`

- Automatically links an Axiom run into the universal spine.
- Recomputes CIDs for all listed artifacts.
- Enforces path safety (no traversal outside repo).

`lgwks run index <run-id-or-path>`

- Prints the structured run index.

## Hardening

- **Path Safety**: Rejects absolute paths or `..` segments in the index.
- **Verification**: All artifact CIDs are recomputed from bytes on adoption.
- **Fail-Closed**: Missing artifacts listed in a child index cause adoption to fail.
- **Idempotency**: Multiple adoptions of the same run are stable.
